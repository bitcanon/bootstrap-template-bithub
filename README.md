# BitHub Bootstrap Template
An HTML template using Bootstrap CSS Framework.

## Introduction
This template is created to resemble the GitHub site as a starting point in a project.

Using:
- Bootstrap 4.5.3
- Font Awesome 5.12
- jQuery 3.5.1
